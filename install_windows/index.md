# 安装

重启电脑，战神进入Logo时狂按F2和F7进入`BIOS`，进入BIOS后用U盘来boot。

接下来跟着步骤走就行

# 分区

分区这一块划重点，删掉所有的原来的分区，留下两个未分区的物理盘(机械+固态)，新建主分区，用固态盘作为系统盘(c盘)，**接下来一定要选中刚刚建好的主分区**，开始写入操作系统数据。

接下来就是熟悉的配置系统步骤了。
